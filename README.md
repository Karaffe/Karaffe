notepad-langのJava実装です。メモ帳でなんの問題も無く書ける  
シンプルでパワフルな言語を目指しています。  

###特徴  

* 静的型付け + **強力な**コンパイル時エラーチェック
* 補完やシンタックスハイライト、オートインデントが無いエディタを最大限に考慮した文法
* 遅延評価構文による遅延評価
